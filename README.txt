TeamFlow - Advanced Team Task Manager

Project Name: TeamFlow
Purpose: Full-stack Team Management
Tech: Node.js, Express, React, SQLite

HOW TO RUN:
1. npm install
2. npm run dev
3. Open http://localhost:3000

CREDENTIALS:
Sign up for a new account to experience the full dashboard.

MAIN FEATURES:
- Dashboard stats (live updates)
- Project creation and team invite
- Task kanban board (Drag-update simulation)
- Secure JWT login
- Mobile responsive sidebar
