# TeamFlow - Advanced Team Task Manager

TeamFlow is a production-ready, full-stack task management application designed for modern distributed teams. It features a robust backend powered by Node.js, Express, and SQLite, and a polished frontend built with React, Vite, and Tailwind CSS.

## Features
- **Secure Authentication**: JWT-based auth with password hashing.
- **Project Management**: Create, view, and manage projects with member collaboration.
- **Task Management**: Kanban-style task board with status and priority tracking.
- **Dashboard Analytics**: Real-time stats on projects, tasks, and team productivity.
- **Responsive UI**: Dark-themed, professional interface that works on all devices.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express, SQLite3, JWT, bcryptjs.
- **Deployment**: Railway-ready architecture with production builds.

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Configure `.env` based on `.env.example`.
4. Run `npm run dev` to start the development server.
5. Access the app at `http://localhost:3000`.

## Production
Run `npm run build` then `npm start` to run in production mode.
